# マンション購入シミュレーター

マンション購入から売却まで、トータル損益をシミュレーションできるWebアプリケーションです。

## 機能

- **初期費用計算**: 契約時・決済時・後日費用を個別に入力・計算
- **現金/ローン購入切替**: 現金購入とローン購入を簡単に切替・比較
- **購入後キャッシュフロー推移**: 管理費、修繕積立金、固定資産税、保険、ローン返済などを年ごとに計算
- **売却シミュレーション**: N年後の売却価格と諸費用から売却手取りを計算
- **居住期間設定**: 任意の期間を居住/非居住に設定可能
- **賃貸運用**: 非居住期間の賃貸収入を計算（空室率・管理費考慮）
- **家賃節約オプション**: 居住期間の「みなし収益」を計上可能
- **統合損益表示**: 購入から売却までのトータル損益とブレークイーブン年を表示
- **グラフ表示**: 年間キャッシュフロー推移と累積キャッシュフロー推移を可視化

## デプロイ方法

### Cloudflare Pages

1. GitHubリポジトリにプッシュ
2. Cloudflare Dashboardで「Pages」→「新しいプロジェクトを作成」
3. GitHubリポジトリを選択
4. ビルド設定：
   - フレームワークプリセット: **なし**
   - ビルドコマンド: **空欄にする（削除）**
   - ビルド出力ディレクトリ: **空欄にする（削除）**
5. 「保存してデプロイ」

**重要**: ビルドコマンドとビルド出力ディレクトリは**空欄**にしてください。このプロジェクトは静的サイトのため、ビルドは不要です。

### ローカル開発

```bash
# リポジトリをクローン
git clone https://github.com/isewanko/apptest.git
cd apptest

# ローカルサーバーで起動（例：Python）
python -m http.server 8000

# ブラウザで http://localhost:8000 を開く
```

## 使用方法

1. **購入方法を選択**: 上部の「現金購入」または「ローン購入」ボタンをクリック
2. **初期費用タブ**: 購入価格や各種費用を入力
3. **購入後キャッシュフロータブ**: 
   - 保有年数と居住期間を設定
   - 年間固定費用を入力
   - 賃貸運用の設定（非居住期間）
   - 家賃節約オプションの設定（居住期間）
4. **売却・トータル損益タブ**: 売却価格と諸費用を入力してトータル損益を確認

すべての入力は即座に再計算され、結果が表示されます。
